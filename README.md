# AI Project Idea Generator

## Overview
AI Project Idea Generator will be a web app that generates high‑quality project ideas for students and builders based on their skills, interests, timeline, and goals.

## Problem It Solves
Finding the right project idea is time‑consuming and often generic. This tool will generate focused, realistic ideas that align with a user’s background and objectives, making it easier to build a strong portfolio.

## Planned Features
- Guided input form (domain, skills, interests, timeline, goal)
- Optional job‑description PDF input
- AI‑generated project ideas with:
  - Title & tagline
  - Tech stack
  - Difficulty rating
  - Key features
  - Step‑by‑step plan
- Expandable, readable project cards

## Tech Stack
- **Frontend:** Python Streamlit  
- **Backend:** Python Flask  
- **AI:** Google Gemini API  

## Project Structure
- `/app.py` — Streamlit UI
- `/backend/` — Flask API 
- `/services/` — AI logic and helpers
- `/styles/` — UI
